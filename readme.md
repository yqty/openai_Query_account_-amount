# OpenAI 计费信息查询
这个项目是一个用来查询OpenAI计费信息的小工具。可以通过提供API Key来查询OpenAI账户的订阅信息和计费信息。

安装
安装所需依赖项，可以通过pip install requests streamlit来安装。
下载代码文件。
使用示例
在终端运行streamlit run 查询账户金额.py启动应用。
在网页上输入您的OpenAI API Key。
点击"查询"按钮。
